# Haven Communities

## Problem
Land acquisition in Nigeria is complex and often lacks transparency. Buyers struggle to find reliable information about available properties, pricing, and legitimate sellers.

## Solution
Haven Communities is a modern real estate platform that simplifies land sales in Nigeria. The platform provides transparent property listings with verified information, educational content about land investment, and direct communication channels with the sales team.

## Key Features
- Property listings with Nigerian Naira pricing
- Educational blog content about land investment
- Newsletter subscription system
- Contact forms for inquiries
- Admin panel for content management

## Technology Stack

**Frontend**
- React 18 with TypeScript
- Vite (build tool)
- Tailwind CSS (styling)

**Backend**
- Supabase (database and API)
- Edge Functions (serverless backend)
- Supabase Auth (authentication)

**Deployment**
- Render (frontend hosting)

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Environment Setup
Copy `.env.example` to `.env` and add your Supabase credentials.
