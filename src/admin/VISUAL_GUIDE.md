### Typography Hierarchy
```
H1 (Page titles):     30px / Avenir Heavy / #155eef
H2 (Section titles):  24px / Avenir Heavy / #181d27
H3 (Card titles):     18px / Avenir Medium / #101828
Body:                 16px / Avenir Regular / #535862
Labels:               14px / Inter Medium / #475467
```

### Color Usage
- **Blue (#155eef):** Buttons, links, headings, icons
- **Green (#039855):** Success states, property stats
- **Red (#E83D3D):** Logout, errors, warnings
- **Gray (#475467):** Body text, labels
- **Light Gray (#eaecf0):** Borders, dividers
- **White (#ffffff):** Backgrounds, cards
